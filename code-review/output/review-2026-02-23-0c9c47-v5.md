### [MEDIUM] PYT-001 — Lock files accumuleren op schijf
**Bestand:** `lib/store.py:72-83` | **Confidence:** 0.90 | **Agent:** python-specialist

De `_session_lock` context manager opent een `.lock` bestand maar verwijdert deze nooit. Over tijd stapelen deze bestanden zich op in `SESSIONS_DIR`. Dit is geen functioneel probleem maar een resource-leak op het filesysteem.
